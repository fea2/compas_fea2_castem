CastemRigidPart

class compas_fea2_castem.model.CastemRigidPart(**kwargs)

Bases: RigidPart

Castem implementation of the RigidPart.

The control of the whole part via a reference point is not implemented yet. To do so, either apply the load on all the nodes composing the cube.

A pist of implementation could be done via the SUPER-ELEMENT amd master nodes. https://www-cast3m.cea.fr/index.php?page=notices&notice=SUPE

Rigid part.Base class for Parts.
